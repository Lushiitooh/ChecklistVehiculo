¿Necesita usar esto en otro dispositivo o navegador?

Los datos se guardan solo en este navegador. Para usarlos en otro dispositivo:

  1. Copie el archivo conversor.html a ese dispositivo (o abra el mismo Excel allá).
  2. Cargue el Excel en el conversor de ese dispositivo.

O bien, use siempre el mismo dispositivo para las inspecciones.
